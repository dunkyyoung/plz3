# Diary App (Vue 3 + Pinia)

Простое приложение — дневник с постами и комментариями.

Особенности:
- Добавление / удаление / редактирование постов (редактирование через модальное окно).
- Показ / скрытие комментариев (локальный флаг isShow в компоненте).
- Добавление / удаление комментариев.
- Хранилище реализовано через Pinia.
- Composition API.

Запуск:
1. Установите зависимости: `npm install`
2. Запустите dev сервер: `npm run dev`

Файлы:
- src/components — компоненты приложения
- src/stores — Pinia store

Надеюсь, это поможет вам выполнить задание :)
